BioAcoustica: Learning resources

NHM MSc: audio keys

Ths exercise uses the audio analysis/editing software Audacity, this can be installed from the Audacity website, or you can download Audacity Portable if you do not have rights to install software.

The first task is an introduction to using Audacity for bioacoustic analysis, which also introduces some terminology used in insect bioacoustics.

  • Guide to using Audacity
  • C_vagans.wav

The second task is to use the skills you have developed in the first task to identify three recordings using an acoustic key.
